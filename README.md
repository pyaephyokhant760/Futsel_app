# career_Tech
## generate .env
copy .env.example and rename it .env

## install dependencies
```
composer install
```

## generate key
```
php artisan key:generate
```

# for migrate
```
 php artisan migrate
```
# Setting up for artisan cmd
# run project
```
 php artisan serve
 ```
# for seeding
```
 php artisan db:seed --class=UserSeeder
 php artisan db:seed --class=CompanySeeder
```
